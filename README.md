# Sales Analysis Project

This is a beginner-level sales insights capstone project using Python and its libraries like pandas and matplotlib.

## Project Objective
The objective of this project is to analyze sales data and generate useful business insights such as revenue growth, customer analysis, and product performance.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Questions Solved
- Total Revenue
- Month over Month Growth
- Year over Year Growth
- Most Ordered Time Window
- Customer Lifetime Value
- Customer Churn Rate
- Average Order Value
- Sales per Region / Category / Product
- Declining Products
- Product vs Month Evaluation

## Graphs Included
- Sales Per Region
- Sales Per Category
- Sales Per Product
- Orders by Hour
- Month-wise Revenue Trend
- Year-wise Revenue Trend
- Highest Ordered Product Each Month

## Files Included
- sales_analysis.ipynb
- Dataset Excel File

## Learning Outcomes
Through this project, I learned:
- Data cleaning using pandas
- Data analysis using groupby()
- Data visualization using matplotlib
- Working with Jupyter Notebook
- Extracting business insights from sales data
